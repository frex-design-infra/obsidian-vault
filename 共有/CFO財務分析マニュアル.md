- 資金繰り悪化の兆候

## 優秀な財務分析とは

### 1. 数字の正確性（基本中の基本）
残高・入出金・利益すべて正確。計算ミスは信頼を失う。

### 2. タイムリーな報告
月次・週次・日次で状況把握。問題の早期発見。

### 3. 経営判断に使える分析
単なる数字の羅列ではなく、「だから何をすべきか」まで示す。

### 4. キャッシュフロー重視
利益が出ていても資金ショートしたら倒産。「いつ入金？いつ支払？」を常に把握。

### 5. 予実管理
予算と実績の差異分析。計画とズレた理由を特定。

### 6. リスク察知
資金繰り悪化の兆候を見逃さない。借入タイミングの最適化。

## 技術メモ

### Node.jsでの計算例
```javascript
const initBalance = 17186079; // 初期残高
let balance = initBalance;

data.forEach(entry => {
  const amount = entry.amount || 0;
  const type = entry.type;
  
  if (type === 'in') {
    balance += amount;
  } else if (type === 'out') {
    balance -= amount;
  }
});

console.log(`4月末残高: ¥${balance.toLocaleString()}`);
```

### 重要な注意点
- 一括取得（全データをdate範囲で取得）は取得漏れのリスクあり
- 必ず各口座ごとに `account=eq.<口座ID>` で個別取得
- `type` フィールドで入出金を判定（amount自体は常に正の数）
