anon keyはブラウザに保存されます。チームで同じURLとキーを使用します。

Supabase で実行するSQL（初回のみ）
create table public.bridges ( id uuid default gen_random_uuid() primary key, management_number text, name text, data jsonb not null, created_by uuid references auth.users(id), created_at timestamptz default now(), updated_at timestamptz default now() ); create table public.approvals ( id uuid default gen_random_uuid() primary key, bridge_id uuid references public.bridges(id) on delete cascade, user_id uuid references auth.users(id), user_email text, notes text, approved_at timestamptz default now() ); alter table public.bridges enable row level security; alter table public.approvals enable row level security; create policy "bridges_select" on public.bridges for select to authenticated using (true); create policy "bridges_insert" on public.bridges for insert to authenticated with check (true); create policy "bridges_update" on public.bridges for update to authenticated using (true); create policy "approvals_select" on public.approvals for select to authenticated using (true); create policy "approvals_insert" on public.approvals for insert to authenticated with check (auth.uid() = user_id);

使い方

  1. セットアップ：template.xlsx を読み込む
  2. インポート：過年度点検Excelファイルを複数ドロップ
  3. 確認・編集：マッピングされたデータを確認・修正
  4. 写真管理：橋梁ごとに写真フォルダを選択、サムネイルで確認・番号割当
  5. エクスポート：国総研様式Excelを橋梁ごとに出力（ZIP）
損傷図PDF連携は後日実装予定。現在は写真番号を手動で割り当ててください。
