# Thinq — UI設計

> 「黒い宇宙に、心の波が流れていく。」

← コンセプト全体は [[Thinq 01 - アプリコンセプト]] へ

---

## 設計の大原則

| 原則 | 内容 |
|------|------|
| **黒が基本** | 深い黒背景。色彩が最大限に映える |
| **波形がコンテンツ** | UIパーツは最小限。波が主役 |
| **静けさ** | ボタンは少なく、文字は少なく、余白を大切に |
| **匿名の徹底** | アバター・名前・アイコン一切なし |
| **評価UIゼロ** | いいね・コメント・フォローボタンは存在しない |

### 画面1：タイムライン「心の海」

> みんなの波が流れる。ただ眺める。

```
┌─────────────────────────────┐  ← 黒背景
│                             │
│  thinq                      │  ← ロゴのみ（極小）
│                             │
│  ~~~波形A（ティール）~~~~~   │  ← 穏やかな低波
│                             │
│  ///波形B（コーラル）/////   │  ← 高い鋭い山
│                             │
│  ---波形C（インディゴ）----  │  ← 深く沈んだ波
│                             │
│  ~~~波形D（アンバー）~~~~~   │  ← 広く大きな波
│                             │
│  ・・・・                   │  ← 無限スクロール
│                             │
└────────────────────────────┘
      🌊        ✏️        〜
```

**動き・インタラクション**
- 波は緩やかにアニメーションしている（静止していない）
- 上から下へ流れる。新しい投稿が上に追加される
- 波をタップ → 画面5（日記詳細）へ
- 無限スクロールで過去の波を遡れる

**表示する情報**
- 波形のみ（色・形）
- 投稿からの経過時間（「2時間前」）
- 👁️ ビュー数（小さく表示）
- ユーザー情報は一切なし

### 画面2：書く画面

> ただ書く。1文字でも、1000文字でも。

**投稿スタイルは問わない。つぶやきでも日記でも同じ画面。**

```
┌─────────────────────────────┐
│                             │
│  今、何を感じていますか？    │  ← プレースホルダー（消えるだけ）
│                             │
│  ┌─────────────────────┐   │
│  │                     │   │
│  │  テキスト入力エリア   │   │  ← 上限なし（つぶやき〜日記）
│  │                     │   │  ← 1行でも数百行でもOK
│  │                     │   │
│  └─────────────────────┘   │
│                    [34文字]  │  ← 文字カウンター（上限なし）
│                             │
│         ┌──────────┐        │
│         │  流す →  │        │  ← 投稿ボタン
│         └──────────┘        │
│                             │
│  🎙️ 話して入力              │  ← 音声入力オプション
│                             │
└─────────────────────────────┘
      🌊        ✏️        〜
```

**投稿スタイル別の体験**

| スタイル | 文字数 | 波形の特徴 |
|---------|--------|-----------|
| つぶやき | 1〜100文字 | シンプルな波、色1〜2色 |
| メモ・日記 | 100〜1000文字 | 複雑で豊かな波、色が混ざり合う |

**動き・インタラクション**
- 入力中、背景に薄く波形のプレビューがうっすら見える
- 「流す →」ボタンをタップ → 画面3（波形生成）へ
- 投稿後は編集・削除不可（リアルな瞬間を残す哲学）
- キーボードが出たら「流す」ボタンがキーボード直上に浮く
- 1日に何度でも投稿できる（つぶやき感覚でOK）

### 画面3：波形生成画面（AI分析中）

> あなたの心を、読んでいます。

```
┌─────────────────────────────┐
│                             │
│                             │
│                             │
│    あなたの心を              │
│    読んでいます...           │  ← フェードイン
│                             │
│                             │
│    ～～～～～～～～～～～     │  ← 波形がゆっくり生成される
│                             │
│                             │
│    あなたの心の色は          │  ← 分析完了後に表示
│                             │
│    ● 穏やか × 少しの孤独    │  ← 感情ラベル
│                             │
│    ──── 心の海に流します ──── │  ← カウントダウン or タップ
│                             │
└─────────────────────────────┘
```

**動き・インタラクション**
- 投稿テキストをClaude APIに送信してanalysis中
- 波形がじわじわと色づきながら生成されるアニメーション
- 感情ラベルがフェードインで表示
- 3秒後 or タップで、波形がタイムラインへ流れていくアニメーション
- そのままタイムラインへ遷移

### 画面5：日記詳細

> 誰かの心に、触れる。

```
┌─────────────────────────────┐
│  ← 戻る                      │
│                             │
│  ────────────────────────── │
│                             │
│  ~~~波形（大きく表示）~~~~~   │
│  ～〜〜〜〜〜〜〜〜〜〜〜〜〜  │  ← 緩やかにアニメーション
│                             │
│  ● 穏やか × 少しの孤独      │  ← 感情ラベル
│                             │
│  ────────────────────────── │
│                             │
│  今日は仕事が終わって帰り道、 │
│  ふと空を見上げたら、        │  ← 日記テキスト
│  思ったより綺麗な夕焼けで、  │
│  少し泣きそうになった。      │
│                             │
│  ────────────────────────── │
│                             │
│  2時間前   👁️ 847            │  ← 時間とビュー数のみ
│                             │
└─────────────────────────────┘
```

**動き・インタラクション**
- 波形は画面上部に大きく表示され、緩やかに揺れている
- テキストは下にスクロールして読める
- 「戻る」以外のアクション一切なし（コメント・いいねボタンなし）
- この画面を開くとビュー数が+1される
- 閲覧者のビュー数加算は1投稿につき1回のみ（重複なし）

## オンボーディング（初回起動）

```
画面A：                    画面B：                    画面C：
「Thinqへようこそ。」       「書いた言葉が              「誰かが見る。
                           心の色になります。」         それだけでいい。」
  ── 次へ ──                ── 次へ ──                 ── はじめる ──
```

- 3画面のみ。シンプルに
- アニメーション波形がデモとして流れる
- アカウント登録不要。「はじめる」をタップしたら即タイムライン

## アニメーション設計

| 場所 | アニメーション | 技術 |
|------|-------------|------|
| タイムライン波形 | 緩やかに揺れる（sin波） | Flutter CustomPainter |
| 波形生成画面 | じわじわ色づきながら出現 | AnimationController |
| 投稿→タイムライン | 波がフロートして合流 | Hero Animation |
| 画面遷移 | フェード（左右スライドなし） | FadeTransition |
| ビュー数 | タップ時にカウントアップ | AnimatedSwitcher |

## アイコン・UIパーツ

- ボトムナビゲーションアイコン：線画のみ（Filled不使用）
- 「流す →」ボタン：角丸なし、細い枠線のみ
- 戻るボタン：← のみ（テキストなし）
- ビュー数：👁️ + 数字（絵文字ではなくカスタムアイコン推奨）
