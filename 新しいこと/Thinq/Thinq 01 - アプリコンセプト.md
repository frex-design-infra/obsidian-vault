# Thinq — アプリコンセプト v3.0

> 「書くだけで、自分だけのAurora Ringが育っていく。」

---

## ひとことで言うと

匿名の日記・メモを書くと、AIが心理学・哲学をベースに感情状態を分析し、  
**カラフルなリボン波形として可視化するコミュニティプラットフォーム。**

書くたびに「自分だけのAurora Ring」が育ち、みんなの波が心の海に流れる。  
コメントなし。評価なし。ただ「見てもらえた」という静かな共感だけがある。

---

## なぜ作るのか

SNSは疲れる。いいねが欲しくて、コメントが怖くて、本音が書けない。  
でも人は誰かに「読んでもらいたい」という気持ちを持っている。

Thinqは、**評価されない場所で本音を書く**ためのプラットフォーム。  
書いた言葉はAIに読まれ、心の色になり、静かに世界に流れていく。  
そして書くたびに——**自分だけのAurora Ringが、少しずつ育っていく。**

```
日記・メモ・つぶやきを書く（匿名）
        ↓
AIが心理学・哲学ベースで感情状態を分析
        ↓
リボン波形が生成されタイムラインに流れる
        ↓

        ↓

        ↓

コメントはない。ただ「見てもらえた」
```

## 「育てる」という核心体験

### Aurora Ringとは

自分の投稿履歴から生成される、**自分だけの感情の肖像画。**  
書くたびに感情スコアが蓄積され、リングの形が変わっていく。

```
1日目
  細く、スパイクが少ない。まだ育っていない。

1週間後
  よく書く感情のゾーンが育ってきた。
  自分の「感情のクセ」が見えてくる。

1ヶ月後
  怒りが多かった月 → 右側のコーラルが鋭く高い
  穏やかだった月  → 左側のティールがふわっと広い
  これがあなたの、この1ヶ月の感情の形。
```

### たまごっちとの違い

| たまごっち | ThinqのAurora Ring |
|-----------|-------------|
| 意識して世話をする | **書くだけで勝手に育つ** |
| 可愛いキャラが育つ | **自分の感情そのものが育つ** |
| ゲームとして楽しむ | **自己理解として使う** |
| 死ぬ・リセットがある | **感情の記録は消えない** |

> 「1年後の自分のAurora Ringを見たい」  
> これがThinqを続けるモチベーション。

## ビジュアルデザイン —「心のスペクトル」

### 波形スタイル：リボン波形

- 波の山で**リボンが太くなる**（感情の強さが形に出る）
- 色はグラデーションで左から右へ流れる
- 短文・1色 / 中文・2色グラデーション / 長文・虹色スペクトル

### カラーパレット

- **背景**: インディゴブラック（`#090812`）
- **波形**: 発光するリボングラデーション
- **文字**: ソフトホワイト（`#f0eeff`）・明朝体

### 感情 × 色のマッピング（Russellの環状モデルベース）

```
        ↑ 高覚醒（High Arousal）
  🟡 ワクワク・喜び  |  🔴 怒り・不安
                    |
快 ─────────────────┼───────────────── 不快
                    |
  🟢 穏やか・安心   |  🔵 悲しみ・孤独
        ↓ 低覚醒（Low Arousal）
```

| 感情 | 波の色 | リボンの特徴 |
|------|--------|------------|
| 😤 怒り・激情 | コーラルレッド〜オレンジ | 鋭く高い山・速い流れ |
| 🤩 ワクワク・高揚 | アンバー〜イエロー | 広く大きな波 |
| 😊 喜び・感謝 | ライムグリーン〜ゴールド | 穏やかで温かい波 |
| 😌 穏やか・安心 | ティールグリーン | ゆったりした低波・遅い流れ |
| 😰 不安・緊張 | ローズ〜マゼンタ | 細かく震える・速い流れ |
| 😔 悲しみ・孤独 | ブルー〜パープル | 深く沈んだ波・遅い流れ |
| 🌫️ モヤモヤ・疲れ | スチールブルー〜グレー | 霞のような低い波 |
| 🌀 複雑・混乱 | インディゴ〜バイオレット | 不規則な重なり |

## AI感情分析の設計

### 参照する心理学・哲学的根拠

| 理論 | 活用方法 |
|------|----------|
| **Russellの環状モデル** | valence（快・不快）× arousal（覚醒度）で感情を2D座標に配置 |
| **Plutchikの感情の輪** | 8基本感情とその混合・強度グラデーション |
| **フロイトの防衛機制** | テキストに見られる「抑圧」「投影」などのパターン検出 |
| **ヘーゲルの弁証法** | 対立する感情が共存している状態の検出 |
| **仏教・ストア哲学** | 「受容」「手放し」「現在への集中」の言葉パターン |
| **認知的不協和理論** | 「わかってるけど…」という葛藤表現の検出 |

### 分析フロー

```
テキスト入力
    ↓
Claude API（感情分析プロンプト）
    ↓
valence スコア（-1.0 〜 +1.0）
arousal スコア（0.0 〜 1.0）
感情強度スコア（0.0 〜 1.0）
主要感情カテゴリ（複数可）
    ↓
① リボン波形パラメータに変換 → タイムラインに流す
② 感情スコアをAurora Ringに蓄積  → 自分のAurora Ringが育つ
```

### 混在感情の表現

人の感情は1つとは限らない。「嬉しいのに悲しい」「怖いけど楽しい」——複数の感情が同時に存在することを、Thinqはそのまま可視化する。

| 検出感情数 | 波形の色 | 例 |
|-----------|---------|-----|
| **1感情** | 単色リボン | 穏やか → ティールグリーン一色 |
| **2感情** | 2色グラデーション | 喜び＋不安 → ゴールドからマゼンタへ |
| **3感情以上** | スペクトル（虹色） | 喜び＋悲しみ＋怒り → 虹色に輝く |

Aurora Ringでも同様に、複数のゾーンが同時にスパイクを伸ばすことで「混在」が視覚的に現れる。

> 「嬉しいのに泣けた」——その複雑さをそのまま、色にする。

## 機能設計

### 書く
- テキスト入力（日記・メモ・つぶやき）
- 1日何回でもOK（つぶやき〜長文日記まで）
- 音声入力（話して入力）
- 完全匿名（名前・アカウント不要）
- 投稿後は編集不可（リアルな瞬間を残す）

### タイムライン「心の海」
- みんなのリボン波形が左へ流れていく
- 短文は1色・長文は虹色スペクトル
- 波をタップ → 日記詳細へ
- ビュー数のみ表示（コメント・いいねなし）

### 日記詳細 — 書いた人のAurora Ring

波をタップして日記を読んだとき、その人の**Aurora Ringがプロフィールとして表示**される。

- 名前なし・アイコンなし・フォロワー数なし
- **Aurora Ringだけが、その人を語る**
- 「このリング、いつも青が深い人だな」という静かな共感が生まれる
- 匿名なのに"人"を感じられる、Thinqだけの体験

> 名前を知らなくても、リングの形でその人の感情の歴史が伝わる。  
> Aurora Ring = Thinqにおける「顔」であり「アイデンティティ」。

### 自分の波「育てる」
- **自分だけのAurora Ring**（投稿するたびに育つ）
- 感情の割合バー（今月は何が多かったか）
- 投稿履歴リスト
- Aurora Ringは1年後も消えない・成長し続ける

### インタラクションの哲学
- コメントなし
- いいねなし
- フォローなし
- **ビュー数だけ** — 「見てもらえた」という静かな事実

### プライバシー設定

他人の感情が気になる・影響を受けやすい人向けのオプション。

| モード | 内容 |
|--------|------|
| **心の海モード**（デフォルト） | みんなの波がタイムラインに流れる |
| **自分の波だけモード** | タイムラインに自分の投稿のみ表示 |

- 自分の投稿は引き続き「心の海」に流れる（発信はON）
- Aurora Ringの育成は両モードで通常通り継続
- いつでも切り替え可能・設定に罪悪感なし

> HSPや、他者の感情に流されやすい人にとって、  
> 「自分のAurora Ringだけを育てる場所」として使える。

### 波の共鳴

タイムラインを流れる波のうち、**自分のAurora Ringと感情パターンが近い人の波**だけ、ほんのわずかに輪郭が光る。

- 通知なし・コメントなし・いいねなし
- ただ、光る
- 「世界のどこかに、似た心を持つ人がいる」——その静かな事実だけが伝わる
- 自分の波だけモード（プライバシー設定）の人の波は共鳴しない

> フォローもリプライもいらない。  
> 似た感情を持つ誰かと、そっとつながれる。Thinqにしかない体験。

### リングの夜明け

書いた日はリングが輝く。書かない日が続くと、リングは**静かに眠っていく**。

```
書いた日    → Aurora Ringが輝く
1日後       → 少し落ち着く
3日後       → スパイクが短くなる。でも消えない
7日後       → 静かに、ゆっくりと呼吸しているだけ

また書いた日 → 夜明けのように、ゆっくりと光が戻ってくる
```

- リングは**死なない**。眠るだけ。（たまごっちとの決定的な違い）
- 継続へのプレッシャーではなく、**「また会えた」という感覚**
- 久しぶりに開いた日に、静かに輝きを取り戻すリングが迎えてくれる

## 技術スタック

| レイヤー | 技術 | 理由 |
|---------|------|------|
| フロントエンド | Flutter | iOS/Android同時対応・波形アニメーションが得意 |
| バックエンド/DB | Supabase | 無料枠あり・SQL・Auth内蔵・自分でサーバー不要 |
| AI感情分析 | Claude API | テキスト→感情スコアの精度が高い |
| 波形描画 | Flutter CustomPainter + Shader | リボン波形・発光エフェクト |
| アニメーション | flutter_animate | 波のなめらかな流れを実現 |
