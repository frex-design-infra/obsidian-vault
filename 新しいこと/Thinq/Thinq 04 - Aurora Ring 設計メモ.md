# Aurora Ring — 設計メモ

## 概要

Aurora Ring は 200本のスパイクで構成されるリング。  
各スパイクの高さは感情の重みづけ（`MY_W`）から生成される波形で決まる。  
`growth` パラメータ（0 → 1）でリングの「成長度」を表現する。

---

## growth パラメータが制御する4つの要素

| パラメータ | growth=0（芽生え） | growth=1（成熟） | 変化の式 |
|---|---|---|---|
| **maxS**（スパイク最大高） | `side × 0.10` | `side × 0.30` | `side × (0.10 + g×0.20)` |
| **rotSpeed**（回転速度） | `0.20 rad/s` | `0.36 rad/s` | `0.20 + g×0.16` |
| **floor**（スパイク底上げ） | `0` | `maxS × 0.07` | `g × maxS × 0.07` |
| **bAmp**（呼吸の深さ） | `0.04` | `0.18` | `0.04 + g×0.14` |

### 成長ステージのイメージ

```
growth = 0.0  ●  小さく静か、スパイクがほぼ平坦
growth = 0.3  ◉  generate画面。まだ若い、波形が見え始める
growth = 0.5  ◎  中間。回転が感じられ始める
growth = 1.0  ✦  myring画面。完全成長、力強く脈動
```

### 各画面での growth 値

| 画面 | growth | 意味 |
|---|---|---|
| generate | `0.3` | 日記を書いた直後、芽生え |
| detail（author） | `1.0` | 他者のリング、完全成熟 |
| myring | `1.0` | 自分のリング、完全成熟 |
| tab bar icon | 固定値（maxS = S×0.34, bAmp = 0.18） | 常にフル表示 |

## 呼吸（Breathe）

```
breathe = 1
  + sin(tm × 0.55) × bAmp          ← 主呼吸（約1.1秒周期）
  + sin(tm × 0.34 + 1.8) × bAmp×0.65  ← 副呼吸（約1.8秒周期）
```

growth=1 のとき bAmp=0.18 → スパイク高が ±18% 脈動する。  
growth=0 のとき bAmp=0.04 → ほぼ平静、±4% のみ。

---

## 感情ゾーン（makeEnv）

リングを角度方向に7つのゾーンに分割。各ゾーンが1感情を担当。

```
0%  ──── 20%   anger  （怒り）  鋭い多重スパイク  π×18, π×28
18% ──── 36%   anx    （不安）  不規則な3層波    π×15, π×23, π×33
34% ──── 55%   sad    （悲しみ）なだらかな長波    π×5.6, π×14
50% ──── 60%   fat    （疲れ）  重く低い波        π×3.2, π×12
57% ──── 72%   calm   （穏やか）等間隔の波         π×4.4, π×16
68% ──── 86%   joy    （喜び）  弾む明るい波       π×7.6, π×16
82% ──── 99%   excite （興奮）  密な高速3層波     π×10, π×18, π×28
```

- `zW(t, start, end)` : ゾーンのウィンドウ関数（ゾーン外はゼロ）
- `sn(x)` : `sin²(x) × sign(sin(x))` → ゼロ付近が急峻なサイン波
- ピーク値 `pk = max(ang, anx, sad, fat, clm, joy, exc)`
- 最終値 = `max(0.08, pk × breathe)` （最低ラインあり）

## smoothBuf（ガウス平滑化）

```
カーネル幅 K = ±4（9点）
decay = 0.60  → exp(-k² × 0.60)
```

decay=0.60 はタイトめの設定で細かい山を潰さず保持する。  
（decay が大きいほど幅が狭くシャープになる）

---

## スパイク描画（2パス）

```
Pass 1（bloom）  lineWidth=3.0,  alpha=0.06  ← 淡いグロー
Pass 2（core）   lineWidth=1.0,  alpha=0.88  ← くっきりした芯
```

各スパイクは内円（R）から外側に `h` ピクセル伸びる。  
`R = side × 0.26 × (1 + sin(tm×0.52) × 0.026)` ← 内円もわずかに脈動

---

## レンダリングパイプライン

```
makeEnv(MY_W) → envFn(t, tm)
                    ↓
           smoothBuf() → smooth[200]
                    ↓
          floor 底上げ → breathe 脈動
                    ↓
     renderRing() → bloom pass → core pass → base ring
```

---

## MY_W（自分の感情重みづけ）

```javascript
{ anger:0.42, anx:0.96, sad:0.68, fat:0.80,
  calm:0.88,  joy:0.82, excite:0.65, grat:0.70 }
```

不安(anx=0.96)が最も高く、穏やか(calm=0.88)と喜び(joy=0.82)も育っている。  
→ リングの形が「不安ゾーンが突出しつつも、右半分に穏やかさと喜びが広がる」形状になる。

## 投稿数 → エフェクト解放マイルストーン

`ramp(postCount, lo, hi)` で各エフェクトが段階的に解放される。

| 投稿数 | エフェクト | 変化の内容 |
|---|---|---|
| **1投稿** | gAura 開始 | 外側オーラ霧がうっすら現れる |
| **1投稿** | gDisk 開始 | 内側ディスク光がうっすら現れる |
| **10投稿** | gBloom 開始 | スパイクのグロー光が強まり始める |
| **30投稿** | 内輪② 開始 | R×0.44 の内輪がうっすら（逆回転） |
| **60投稿** | gAura MAX | 外側霧が最大輝度に達する |
| **80投稿** | gDisk MAX | 内側ディスク光が最大輝度に達する |
| **80投稿** | gCore 開始 | 中心コア光（白～紫）が薄く現れ始める |
| **100投稿** | gBloom MAX | スパイクグロー最大（bloom lineWidth=8） |
| **150投稿** | 内輪② MAX | 内輪② が最大輝度で安定回転 |
| **160投稿** | 内輪① 開始 | R×0.70 の外内輪がうっすら（順回転） |
| **200投稿** | gCore MAX | 中心コア光が最大 |
| **200投稿** | gWhite 開始 | 白芯ライン・外縁ホワイトグロー出現 |
| **250投稿** | 内輪① MAX | 外内輪が最大輝度 |
| **300投稿** | gWhite MAX | 全エフェクト完全成熟・白芯くっきり |

### 各エフェクトの `ramp` 設定

```javascript
const growth   = ramp(postCount,  0, 300); // 基本成長（サイズ・速度・呼吸）
const gAura    = ramp(postCount,  1,  60); // 外側オーラ霧
const gDisk    = ramp(postCount,  1,  80); // 内側ディスク光
const gBloom   = ramp(postCount, 10, 100); // スパイクグロー
const gInner2  = ramp(postCount, 30, 150); // 内輪② R×0.44
const gCore    = ramp(postCount, 80, 200); // 中心コア光
const gInner1  = ramp(postCount,160, 250); // 内輪① R×0.70
const gWhite   = ramp(postCount,200, 300); // 白芯・外縁白ライン
```

### 各画面での postCount 値

| 画面 | postCount | 意味 |
|---|---|---|
| generate | `90` | 若いリング（gBloom開始済み、内輪②なし） |
| myring | `ringCount`（後述） | 300でリセットするサイクル値 |
| detail（author） | `300` | 他者のリングは常に完全成熟 |
| tab bar icon | 固定（maxS×0.34） | 常にフル表示 |
| Thinqing button | `150` | 内輪②MAX 直前、中間成長 |

## 300投稿サイクル & バッジシステム

リングは **300投稿で1サイクル**。301投稿目からまた芽生えに戻る。

```javascript
// 1サイクル = 300投稿
ringCount = MY_POSTS.length === 0
  ? 0
  : ((MY_POSTS.length - 1) % 300) + 1
// 例: 1→1, 300→300, 301→1, 600→300, 601→1
```

### バッジ（達成の証）

300投稿ごとに達成バッジが付与される。  
バッジはタブバー Ring アイコンの **静的スナップショット**（20px）。  
`.ring-stats` エリア（右下）の total 数字の上に横並びで表示。

```
サイクル数 = Math.floor(MY_POSTS.length / 300)
```

| 総投稿数 | ringCount | バッジ数 |
|---|---|---|
| 0〜299 | 0〜299 | 0 |
| 300〜599 | 300→1〜299 | 1 |
| 600〜899 | 300→1〜299 | 2 |
| 900〜 | … | 3以上 |

## 休眠システム（7日ルール）

投稿が途絶えると、リングが「眠り」に入る。

```
daysSince = today - 最終投稿日
decay     = clamp((daysSince - 7) / 21, 0, 1)  // 7日でfade開始、28日でMAX
vitality  = 1 - decay × 0.70                    // 最低30%の活性を維持
```

### vitality が制御する4つのパラメータ

| パラメータ | vitality=1.0（活性） | vitality=0.30（休眠） |
|---|---|---|
| **maxS**（スパイク高） | `side×(0.10 + growth×0.20)` | growth部分が70%縮小 |
| **rotSpeed**（回転） | `0.20 + growth×0.16` | min 35%まで減速 |
| **bAmp**（呼吸） | `0.04 + growth×0.14` | min 25%まで減衰 |
| **globalAlpha**（全体輝度） | `1.0` | `0.28 + vitality×0.72` |

### 休眠ステージのイメージ

```
daysSince = 0〜6   🌟 通常  — 完全活性
daysSince = 7      😴 休眠開始 — 徐々に静かになる
daysSince = 14     🌙 軽眠  — 回転遅め、少し暗め（vitality≈0.77）
daysSince = 21     💤 深眠  — かなり静か、薄暗い（vitality≈0.53）
daysSince = 28+    ❄️ 冬眠  — 最小限の脈動のみ（vitality=0.30）
```

投稿を再開すると vitality = 1.0 に即時回復。
