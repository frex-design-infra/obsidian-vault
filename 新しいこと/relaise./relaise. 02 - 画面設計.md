# relaise. — 画面設計

---

## 画面構成

```
NOTE  |  LYRICS  |  RE:LAY
```

### NOTE
**自分だけの言葉を、ただ溜め込んでいく。**

- クローズド。誰にも見せなくていい
- 思いついた言葉・フレーズを書き留めておく場所
- 自分の言葉が積み重なっていく
- ここから **Post** して世界に放つ

### LYRICS
**みんなの Post が流れるタイムライン。**

- オープン。自分と他のユーザーの Post が流れる
- X（旧Twitter）のようなフィード形式
- ここから他の人の言葉に **Re:lay** できる

### RE:LAY
**自分の活動記録。**

- `Post` タブ：自分が起点になったリレー
- `Re:lay` タブ：誰かのリレーに参加したもの

---

## 言葉の流れ

```
NOTEに書き溜める
「夕暮れの、名前のない色」
        ↓
Post して LYRICS へ放つ
        ↓
見知らぬ誰かが Re:lay する
「それを君は、橙と呼んだ」
        ↓
またさらに誰かが……
        ↓
作者が Fine にして完結
```

**「Post」** という動作が、このアプリの一番エモい瞬間。
自分の中で温めていた言葉が、世界に出ていく。
