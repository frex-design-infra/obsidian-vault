# 💳 カードウォレットアプリ

## コンセプト
各ショップアプリ（GU・ユニクロ・ツルハ等）の**会員証画面へのランチャー**。
ウォレット風UIでカードを一覧表示し、タップすると該当アプリが起動する。

## 開発場所
`/Users/user/Documents/CardWallet/`

## 技術スタック
- **React Native + Expo**（iOS / Android 両対応）
- AsyncStorage（カードデータの保存）
- Linking API（URLスキームでアプリ起動）
